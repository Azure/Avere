Cluster Configuration and Operation

Cluster operations guide documents explain the features of the Avere operating system and give a reference for how to configure settings. Topics include cluster configuration, monitoring, and data management tasks.

Documents for Avere OS 4.8 and later are under development. Please use the version 4.7 documents provided below.

  • Avere OS 4.7
    • Configuration Guide - Detailed information about the configuration controls available in the Settings and Support tabs of the Avere Control Panel, plus overview information about common configuration tasks
    • Dashboard Guide - How to use the Dashboard tab of the Avere Control Panel to monitor and troubleshoot your Avere cluster
    • Data Management Guide - Explains how to use the FlashMove™ and FlashMirror™ features of Avere OS.
     
  • Avere OS 4.6 and earlier
    • Avere OS 4.6 Configuration Guide
    • Avere OS 4.5 Daily Operations Guide (PDF)

For sites without external website access, a PDF or static HTML archive of the current web documentation is available on request.

Avere OS Release Notes

Release notes describe software changes in the latest release. Changes can include new features as well as feature changes and bug fixes. Release notes documents are organized by major version (for example, 4.6) and include information about the latest available release as well as an archive of notes for older releases in that version.

  • Avere OS 4.8 Release Notes (PDF) - currently at version 4.8.9.1
  • Avere OS 4.7 Release Notes (PDF) - currently at version 4.7.3.2
  • Avere OS 4.6 Release Notes (PDF) - currently at version 4.6.2.5

To obtain release notes for older versions of Avere OS, contact Avere Global Services.
