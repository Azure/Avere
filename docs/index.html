Cluster Configuration and Operation

Cluster operations guide documents explain the features of the Avere operating system and give a reference for how to configure settings. Topics include cluster configuration, monitoring, and management tasks.

  • Avere OS 4.7
    • Configuration Guide - Detailed information about the configuration controls available in the Settings and Support tabs of the Avere Control Panel, plus overview information about common configuration tasks
    • Dashboard Guide - How to use the Dashboard tab of the Avere Control Panel to monitor and troubleshoot your Avere cluster
  • Avere OS 4.6 and earlier
    • Avere OS 4.6 Configuration Guide

    • Avere OS 4.5 Daily Operations Guide (PDF)
